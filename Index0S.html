OS Concepts Reference

Resident Set Size (RSS)

The portion of memory occupied by a process that is held in main memory (RAM). This is what you see growing in a memory leak.

Virtual Memory

A memory management technique that provides an "idealized" view of storage to processes, allowing them to use more memory than physically available via swapping.

Slab Allocation

A kernel memory management mechanism that improves performance by pre-allocating caches for specific object types.
